# observabilidade_java
Descrição:
----------
Projeto em Java e Spring Boot usando observabilidade com Prometheus e Grafana.

Dependências:
-------------
O projeto usa as seguintes dependências:
* Spring Boot
* Spring Data JPA
* Spring Security
* Actuator
* Devtools
* Micrometer
* JsonWebToken
* Swagger
* OpenAPI
* MySql
